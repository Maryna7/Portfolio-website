myFlix React app

Within this project, I was responsible for client-side for an app called myFlix based on its existing server-side code. In this app a user has the opportunity to get information about movies, about the details of a movie, about movie's director, information about movie's genre. Also, the user has the opportunity to create an account and save favorite movies to the list of favorite movies.

The technologies used throughout the project

  • HTML, SCSS
  • Bootstrap as a UI library for styling and responsiveness
  • React library
  • Parcel as its build tool
  • React Redux for state management
